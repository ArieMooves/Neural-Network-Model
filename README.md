## Neural-Network-Model
# Neural network models are computational frameworks, inspired by the neural networks of the human brain, that allow computers to make intelligent decisions with reduced human interference. This project sought to show the training of a neural network model designed for CIFAR-10 from a standard baseline, with improvements and simple alterations to allow more accurate learning and comparison. 

Using the baseline model, replace the model's architecture with each of the variations to conclude which model performs the best.
